Файлы автоматической установки linux

На Github я выложил файлы для автоматической установки Debian 10, Centos 7, Centos 8, Centos 9 на GitHub Gists preseed.cfg

Для создания ISO с Debian, который содержит внутри себя файл установки можно использовать скрипт debian-preseeded-iso.sh

А для запуска kickstart установки CentOS и RHEL нужно выложить файл на доступный ресурс, а в GRUB нажать Tab и дописать inst.ks=http://10.32.5.1/ks.cfg
