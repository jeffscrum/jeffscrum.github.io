VAMX. Usefull Commands

  1. Посмотреть masking view symaccess -sid 966 list view

  2. Посмотреть содержимое port-группы symaccess -sid 966 show PG_Task1 -type port

  3. Добавить порты в группу symaccess -sid 966 -type port -name PG_Task1 add -dirport 8e:1,7f:0,10e:0

  4. Посмотреть информацию по диску symdev -sid 966 show 02E5

  5. Посмотреть состояние массива

symcfg -sid 307 list -env_data -v |grep Failed
symdisk list -sid 307 -failed
symevent -sid 307 list
