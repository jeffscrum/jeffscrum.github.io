VMAX. How to recreate emergency token

Note

To access Central Manager off of EMC’s network use url: https://slc.emc.com/psscm

Validate Failed Tokens in Credential Management

  1. Log in to Central Manager using one of the links above.

  2. Select Token Manager > Installed Token Viewer.

  3. Enter the Symmetrix serial number and select Search.

  4. The list of installed token types will be displayed along with their expiration dates. Make a note of the types that have expired. These are the tokens that will need to be replaced.

Validate Tokens on the Service Processor

  1. Log in to the Service Processor using the default user name and password.

  • User: Admin

  • Password: EMC2Admin

  • Domain: Local Host

  1. Select the SSC Key Client icon on the Service Processor desktop.

  2. Select List Installed Tokens. Confirm the tokens listed on the Service Processor match those displayed in Credential Manager.

  3. Select OK.

  4. On the Service Processor in the SSC Key Client in the Secure Service Credentials Tasks screen select Back then Provision Tokens > Next.

  5. Select Disconnected.

  6. Select Export Token Request File. Save the file on your portable USB media device for uploading to Credential Manager. This is an example for the file format: <sn>_KS_<datetime>.enc (e.g., 192604321_KS_20140204131709UTC.enc).

Deleting Tokens on the Service Processor

  1. Return to Central Manager from your laptop.

  2. Go to Token Manager > Token Provisioning Request > Manual Provisioning for Disconnected sites (SP File available) and select Next.

  3. Enter the Symmetrix serial number of the system and select Search.

  4. The “Service Processor File Upload” prompt will appear.

  5. Browse your USB device and select the file saved from running the SSC Key Client on the Service Processor from Step 7 above.

  6. Select Request Delete Transaction.

  7. Select the check box to the left of each token type. This is required to be removed.

  8. Select Get Token File.

  9. The Save file dialog box will appear. Be patient. This may take longer than expected.

  10. Save this file on your portable USB device. Here is an example of the file: <sn>_KC_<datetime>.enc  (e.g., 192604321_KC_20140204_082319.enc).

  11. Select the SSC Key Client icon on the Service Processor desktop and double-click.

  12. Select Provision Tokens > Disconnect > Import.

  13. Browse your USB device and select the file you downloaded in step 10. You have now deleted the expired tokens on the Service Processor.

Delete Tokens in Credential Manager

  1. Log in to Central Manager.

  2. Select Token Manager > Token Request Queue.

  3. Enter the Symmetrix serial number.

  4. A list of deleted tokens will be shown with a red X icon to the left of each token. Select each icon to confirm the tokens have been deleted.

Create and Add New Tokens

  1. On the Service Processor run the SSC Key Client select Next.

  2. Select Provision tokens > Disconnected > Export.

  3. Save this file to your portable USB device.

  4. On Credential Manager select Token manager > Token Provision Request > Manual Token Provisioning for Disconnected Sites ( SP File Available) > Next.

  5. Enter the Symmetrix serial number and select Search.

  6. The Service Processor File Upload prompt will appear.

  7. Browse your portable USB device and select the file saved in step 2 above and select Upload File.

  8. Select the pulldown for the Available Token Type and select the appropriate token type.

  9. Select Add to Queue. Repeat this for each token that was removed.

  10. When all the new tokens have been added to the Token Transaction Queue select Get Token File.

  11. This may take longer than expected. Be patient while the file is generated. Save this file to your local USB device.

  12. On the Service Processor select the SSC Key Client > Next > Provision Tokens > Disconnected > Import. The browser window will appear. Select the file you saved from step 10 above on your USB device. (This will load the new token files to the Service Processor.)

  13. When prompted for your SSC Manual Password select your SSC password. If prompted for a PIN and/or SDTID Password use the last four digits of the Symmetrix serial number.

Final Steps to Confirm Tokens in Credential Manager

  1. Log in to Credential Manager.

  2. Select Token Manager > Token Request Queue.

  3. Enter the serial number of the Symmetrix.

  4. Listed will be the new tokens that were generated. Select the green Check Marks icon next to each token. This will confirm token addition in Credential Manager.

Validate Connectivity

  1. Generate a new credential by logging into Credential Manager.

  2. Select Request a Credential.

  3. Enter the serial number of the Symmetrix and select search.

  4. Select the serial number underlined in the search box.

  5. Set the duration to 2 days and select create.

  6. Make a note of the Credential.

  7. Log off the Service Processor.

  8. Enter the Credential generated in step 6.

  9. Enter your SSC password and SLC Domain.

  10. Once login is verified, log out and close the case.
